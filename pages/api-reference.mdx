# API Reference

Complete list of all TextBubbles API endpoints. Base URL: `https://api.textbubbles.com`

## Messages

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/v1/messages` | [Send a message](/messages/send) |
| `GET` | `/v1/messages` | [List messages](/messages/list) |
| `GET` | `/v1/messages/:id` | [Get message status](/messages/list) |
| `POST` | `/v1/messages/:id/unsend` | [Unsend a message](/messages/send#unsend-a-message) |
| `PUT` | `/v1/messages/:id` | [Edit a message](/messages/send#edit-a-message) |
| `POST` | `/v1/messages/:id/reactions` | [Send tapback reaction](/messages/reactions) |
| `POST` | `/v1/messages/carousel` | [Send image carousel](/messages/carousel) |
| `GET` | `/v1/messages/scheduled` | [List scheduled messages](/messages/scheduled) |
| `DELETE` | `/v1/messages/:id/schedule` | [Cancel scheduled message](/messages/scheduled#cancel-a-scheduled-message) |

## Error Codes

| Code | HTTP Status | Description |
|------|-------------|-------------|
| `UNAUTHORIZED` | 401 | Missing or invalid Bearer token |
| `VALIDATION_ERROR` | 400 | Request body failed validation |
| `INVALID_PHONE_NUMBER` | 400 | Phone number not in E.164 format |
| `NOT_FOUND` | 404 | Resource not found |
| `REPLY_NOT_FOUND` | 404 | Reply target message not found |
| `CHANNEL_NOT_SUPPORTED` | 400 | Operation not supported on channel |
| `NO_CHANNEL_AVAILABLE` | 400 | Recipient not reachable |
| `SMS_NOT_CONFIGURED` | 400 | SMS providers not configured |
| `ALREADY_UNSENT` | 409 | Message already unsent |
| `ALREADY_DELETED` | 409 | Message already deleted |
| `RATE_LIMITED` | 429 | Too many requests |
| `IMESSAGE_SEND_FAILED` | 500 | iMessage provider error |
| `SMS_SEND_FAILED` | 500 | SMS provider error |
| `INTERNAL_ERROR` | 500 | Unexpected server error |
